🎬 Netflix Desktop App

Python Tkinter ve SQLite kullanılarak geliştirilmiş Netflix benzeri masaüstü platformu.

Proje Özellikleri

  • Kullanıcı kayıt ve giriş sistemi
  • Admin paneli
  • Film ve dizi yönetimi
  • Favori sistemi
  • İzleme geçmişi
  • Öneri sistemi
  • SQLite veritabanı

Kullanılan Teknolojiler

Python, Tkinter, SQLite, OOP, GUI Design, SQL Queries
